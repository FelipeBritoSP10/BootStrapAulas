Aulas Extras

Curso de HTML

No curso de HTML, exploraremos os alicerces da linguagem, abrangendo desde os fundamentos básicos até técnicas avançadas de estruturação de páginas web.


No curso, você aprenderá:

  • Fundamentos do HTML para estruturar páginas web;
  • Uso das tags HTML essenciais;
  • Criação e estilização de formulários;
  • Implementação de semântica HTML para melhor acessibilidade.

Curso ideal para iniciantes que desejam entrar no mundo da programação.


Aprimore suas habilidades de desenvolvimento com exemplos práticos e contextualizados, mergulhando no mundo do HTML e aplicando conceitos essenciais em situações reais.

Comprar curso


Aprenda os fundamentos do HTML para criar estruturas de páginas da web.


Curso de CSS

No curso de CSS, você irá explorar as habilidades necessárias para estilizar suas páginas da web de forma atrativa e responsiva.


No curso, você aprenderá:

  • Princípios básicos e avançados de estilização com CSS;
  • Técnicas modernas de layout, incluindo Flexbox e Grid;
  • Estilização de formulários e elementos de interface do usuário;
  • Criação de estilos responsivos para diferentes dispositivos.

Aprimore suas habilidades de design e torne suas páginas web visualmente atraentes e profissionais.

Comprar curso


Explore as habilidades do CSS para estilizar suas páginas da web de forma atrativa.


Curso de JavaScript

No curso de JavaScript, você irá dominar uma das linguagens de programação mais populares e poderosas para desenvolvimento web.


No curso, você aprenderá:

  • Fundamentos da linguagem JavaScript, incluindo variáveis, condicionais e loops;
  • Manipulação do DOM para interagir com elementos HTML e estilizá-los dinamicamente;
  • Trabalho com eventos para criar interatividade em suas páginas;
  • Uso de funções assíncronas para lidar com requisições AJAX e Promises.

Aprimore suas habilidades de desenvolvimento web e torne suas aplicações mais dinâmicas e interativas com JavaScript.

Comprar curso


Domine o JavaScript para adicionar interatividade e dinamismo aos seus sites.


Curso de React

No curso de React, você aprenderá a construir interfaces de usuário modernas e reativas com uma das bibliotecas JavaScript mais populares e poderosas.


No curso, você aprenderá:

  • Conceitos fundamentais do React, incluindo componentes, props e state;
  • Uso de JSX para escrever código JavaScript e HTML de forma integrada;
  • Gestão de estado com o hook useState e efeitos com o hook useEffect;
  • Criação de aplicações SPA (Single Page Applications) com React Router.

Desenvolva suas habilidades em desenvolvimento web front-end e construa aplicações mais poderosas e eficientes com React.

Comprar curso


Explore os conceitos do React JS para construir interfaces de usuário modernas e reativas.


Curso de GitHub

O curso GitHub oferece uma introdução completa à plataforma e suas principais funcionalidades.


No curso, você aprenderá:

  • Como criar e gerenciar repositórios;
  • Colaboração com outros desenvolvedores usando pull requests;
  • Gerenciamento de problemas e projetos;
  • Como contribuir para projetos de código aberto.

Curso ideal para iniciantes e intermediários que desejam dominar o GitHub para gerenciar projetos e colaborar efetivamente.


Aprenda com exemplos práticos e do mundo real para aprimorar suas habilidades de desenvolvimento.

Comprar curso


Aprenda a utilizar o GitHub para hospedar seus projetos, colaborar com outros desenvolvedores e contribuir para projetos de código aberto.




Aprenda a usar o Git para controle de versão e colaboração em projetos de software.


Curso de Git

No curso de Git, você aprenderá a utilizar o sistema de controle de versão Git para gerenciar seus projetos de software de forma eficiente e colaborativa.


No curso, você aprenderá:

  • Princípios fundamentais do Git;
  • Comandos básicos do Git, como commit, branch, merge, e rebase;
  • Trabalho colaborativo usando repositórios remotos e pull requests;
  • Resolução de conflitos e boas práticas de versionamento.

Curso ideal para desenvolvedores que desejam aprimorar suas habilidades de versionamento e colaboração em equipe.
