O que você precisa para as aulas?

  • Conhecimento básico de HTML, CSS e JavaScript;
  • Um computador ou laptop;
  • Conexão à internet estável.

Como serão as aulas via Google Meet?

As aulas serão realizadas de forma virtual através da plataforma Google Meet.

Algumas características das aulas no Google Meet incluem:

  • Conexão rápida e fácil através de um link compartilhado pelo professor.
  • Interação em tempo real com o professor e aluno.
  • Compartilhamento de tela para apresentações, demonstrações e colaboração.
  • Recursos de chat para perguntas e discussões durante a aula.

Antes do início das aulas, os participantes receberão instruções detalhadas sobre como acessar e utilizar o Google Meet.

Projetos

Você irá aprender em torno do curso fazer alguns projetos.

Portifólio Responsivo

Construir um portifólio responsivo usando o Bootstrap para demonstrar suas habilidades. Este projeto irá abordar conceitos como layout responsivo e componentes do Bootstrap.

Loja Virtual

Construir uma loja virtual completa utilizando Bootstrap para criar um layout responsivo e agradável. Este projeto explorará conceitos como modais e componentes de navegação do Bootstrap.

Cards de produtos

Construir cards de produtos com título, descrição, preço e botões. Este projeto ajudará a praticar a criação de layouts de produtos e a adição de botões.

Tela de login

Construir uma tela de login simples e responsiva. Este projeto permitirá aos alunos praticar a criação de formulários de autenticação.
